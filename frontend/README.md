# Christmas Bingo Frontend

This is the frontend application for the Christmas Bingo System, built with Vue 3, TypeScript, and Vite.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
